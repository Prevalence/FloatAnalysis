### FloatAnalysis



​	**此项目用于自动化检测项目中可能存在的浮点数安全性问题（包括控制流变化与数据流错误）**

​	**共有6个步骤**



 * 数值变量符号提取（基于KLEE开发）
 * 控制流静态分析得到攻击输入，利用Step1的输出，求取round式，实现浮点算式约束求解
 * 数据依赖静态分析得到攻击输入，实现对整型数据的分析和调用2中浮点算式约束求解
 * 利用IRRAM库实现源程序自动转化成高精度程序
 * 验证控制流是否发生改变，追踪软件的执行路径
 * 验证数据以来是否发生改变
